How It Works

The status is determined by checking critical infrastructure components:

  • VPC and subnets
  • Terraform state bucket
  • KMS keyring
  • Compute instances
  • Storage buckets
  • Cloud SQL instances

The status is updated every 15 minutes via GitHub Actions.
